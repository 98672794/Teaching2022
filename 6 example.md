
# 6.1 如何將 Python 連接到 Google 表格


## [在Google API Console新增一個專案](https://console.developers.google.com/) <-- 點網址入 新增專案

    寫地區資料
    按選取專案
    新增專案 名隨意
    確定後等 選取專案 轉為你專案名即OK
